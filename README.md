# 3v3 Game

## Setup Instructions

To set up the game, follow the instructions below:

1. **Clone the Repository**:  
   Use the command:
   ```bash
   git clone https://github.com/richsteve17/3v3.git
   ```

2. **Install Dependencies**:  
   Navigate to the project directory and run:
   ```bash
   npm install
   ```

3. **Start the Development Server**:  
   Use the command:
   ```bash
   npm start
   ```
   Your application should now be running on `http://localhost:3000`.

## Game Documentation

### Overview

The 3v3 game is an exciting multiplayer experience that combines strategy and teamwork. Players will engage in various challenges.

### Gameplay Mechanics

- **Teams**: The game features two teams of three players each.
- **Objective**: Each team competes to achieve the highest score by completing various tasks.
- **Controls**: Players use the following controls to interact with the game:  
  - Move: Arrow keys  
  - Action: Spacebar  

### Scoring System

Points are awarded based on the tasks completed and special achievements during the game. The team with the highest score at the end wins!
